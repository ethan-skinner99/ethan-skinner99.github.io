This site is deployed using GitHub Pages. Any push to the `master` branch will automatically update the live site.

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/ethan-skinner99/ethan-skinner99.github.io.git
   ```

2. Navigate to the Hockey website folder:
   ```bash
   cd ethan-skinner99.github.io/Hockey_website
   ```

3. Open `main.html` in your browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   ```
